# LimeChain CFO Interview Deck - Emergency Canva Build Pack

This folder is the source of truth for the final Canva presentation.

## What to build

Create a 14-slide executive presentation using `deck_master.md` as the content source and `canva_build_instructions.md` as the design source.

## Most important instruction

Do not make this look like a generic AI presentation. It should feel like a serious CFO / strategy consulting deck: sparse, confident, dark accents, strong hierarchy, almost no decoration.

## Narrative

Five asks in the brief are not five separate problems. They are breaks in one revenue-to-cash pipeline.

The CFO answer is a 90-day confidence rebuild:

1. Cash first: stop the bleeding and make runway visible.
2. Truth second: create one version of the numbers.
3. Discipline third: make fixed-price growth safe with guardrails.

## Presentation title

Building Confidence

Subtitle:
A practical 90-day plan: cash first, truth second, discipline third.

## Required output

A Canva presentation with 14 slides, using the slide titles exactly as written in `deck_master.md` unless a line break is needed for design.
