# Master Presentation Brief Template

Use this file before building any new executive deck.

The process is:

1. Fill in this brief.
2. Create a deck source file from the completed brief.
3. Reuse the established Canva design style from the LimeChain deck.
4. Build the final deck in Canva.

## 18. Claude build instruction

After the brief is completed, Claude should:

- Read the completed brief.
- Reuse the established Canva style from the LimeChain deck.
- Build a serious executive presentation.
- Keep one idea per slide.
- Reduce text into diagrams, cards, tables and timelines.
- Preserve the narrative and do not invent unsupported facts.
- Generate the Canva deck from the final deck source file.
